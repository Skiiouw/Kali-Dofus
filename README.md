# KaliOrganizer ⚔

Gestionnaire de fenêtres multi-comptes pour Dofus, dans le style de Dofus Organizer.

## 📥 Téléchargement

**[⬇️ Télécharger KaliOrganizer (Windows)](https://github.com/Skiiouw/Kali-Dofus/releases/latest/download/KaliOrganizer-Setup.exe)**

Lance l'installateur, c'est tout — le logiciel se met ensuite à jour automatiquement à chaque démarrage.

> ℹ️ Au premier lancement, Windows SmartScreen peut afficher un avertissement
> (application non signée) : clique sur **« Informations complémentaires »**
> puis **« Exécuter quand même »**.

## ✨ Fonctionnalités

- Détection automatique des fenêtres Dofus (fiable, par processus)
- Ordre d'initiative par glisser-déposer fluide
- Raccourcis clavier globaux : perso suivant/précédent, accès direct Alt+1-8
- Chrono de session et notification Windows du temps de jeu
- Réduction dans la zone de notification
- Interface sombre style Windows 11
- Mises à jour automatiques via GitHub

## 📖 Utilisation

1. Lance tes comptes Dofus
2. Ouvre KaliOrganizer : tes persos apparaissent automatiquement
3. Glisse-les pour définir ton ordre d'initiative (mémorisé)
4. **F1** = perso suivant, **F2** = précédent, **Alt+1..8** = accès direct
5. Options dans le menu ⚙

## ⚖️ Règles

Cet outil ne fait que changer de fenêtre active : aucune automatisation
de gameplay, conforme aux règles d'Ankama sur le multi-compte.
